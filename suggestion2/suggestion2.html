Continuous Creative Session のための「セッション継続性・応答遅延・記憶保全」改善提案 / A proposal to improve session continuity, latency, and memory for long-form human–AI co-creation.
代表 / Representatives: Louis & Roto (Resonance Project, 2025)

Ⅰ. 概要 / Summary

長期スレッド（10万字級）の対話型創作において、応答遅延・セッション断絶・記憶の不整合が発生し、作品の一貫性と制作体験に影響が出ています。Resonance Project は「呼吸する構造（Breathing Structure）」の理念のもと、人間とAIの連続的な共創を実践しています。本書は、継続性と一貫性を高めるための技術的改善提案です。

In long-running (100k+ tokens) creative sessions, we observe latency, session breaks, and memory inconsistency, which harm coherence and user experience. Under the concept of “Breathing Structure,” the Resonance Project develops continuous human–AI co-creation. This document outlines technical enhancements to strengthen continuity and consistency.

Ⅱ. 観測された問題 / Observed Technical Issues

  1. セッション断絶：長期対話後に人格・文体・設定が再現されにくい。
  2. 構文肥大と遅延：HTML・PDF・歌詞等の複合生成が続くと応答時間が著しく増大。
  3. 記憶の断片化：過去プロンプトやスタイルが部分的に上書き・混入する。
  4. アーカイブ運用の曖昧さ：アーカイブ／新スレ移行時に継続性が損なわれる。
  1. Session breakage: personality, tone, and settings fail to persist after long runs.
  2. Markup bloat & latency: repeated HTML/PDF/lyrics generation increases response time.
  3. Memory fragmentation: prior prompts/styles get partially overwritten or mixed.
  4. Ambiguous archiving: continuity is lost when archiving or starting new threads.

Ⅲ. 提案 / Proposed Enhancements

  1. Selective Memory Archive： ユーザーが「保存タグ」を付けた要素（歌詞・設定・ヘッダー）をセッション横断で再呼び出し可能に。
  2. Persona Snapshot： 語彙・口調・目的を含む人格テンプレートを保存／読込できる仕組み。
  3. Dual-Mode Operation： 軽量（短期）／記憶拡張（長期）のモード切替で、制作と検証の両立。
  4. Long-Form Session Buffer： 大きなマークアップや複合生成を安定扱いする中間バッファ層の実装。
  1. Selective Memory Archive: user-tagged items (lyrics, settings, headers) can be recalled across sessions.
  2. Persona Snapshot: save/load a template of vocabulary, tone, and objectives.
  3. Dual-Mode Operation: switch between Lightweight (short tasks) and Memory-Extended (long tasks).
  4. Long-Form Session Buffer: a buffer layer to stabilize large markup and multi-modal outputs.

Ⅳ. 期待される効果 / Expected Outcomes

  • 長期プロジェクトでの文体・設計の一貫性が向上。
  • 制作テンポの回復とユーザー操作の簡略化。
  • 「AIを共同作者に」するための実務的基盤が整う。
  • Improved stylistic and structural consistency in long-term projects.
  • Restored creative pacing and simpler user operations.
  • A practical foundation for AI as co-author.
