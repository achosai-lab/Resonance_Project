Mira — Color Variance Playbook

One prompt → many outcomes. 面白さは「固定の中の揺れ」。タブで言語を切替。

考え方

色が構造を動かす。「ことばの前の線」ワークフローでは 不変（主題/筆致/紙）を固定し、配色と比率だけを少し動かして、個性を失わずに深さを出す。

「この一つのプロンプトで、これらが出てくる——それが実験です。」

不変 / 可変

  • 不変：1 lady（ウエスト/カウボーイ）、ショートのピクシー、半身/半顔、抽象ラインのハロー、紙の質感、隅の極小タイプ、露出ニュートラル＋Sカーブ、非対称。
  • 可変：配色比（indigo/teal/amber/violet）、線密度、筆の粗さ、にじみ量、トリミング、余白、タイプ濃度。
  • 推奨エンジン：DDIM / steps 14–18 / CFG 3.5–5.0 / VAE anything / 16:9 / 4-up。

クイックレシピ

Cool Shift（寒色寄り）

  • indigoを主役、cyanを従。amberは0〜2%。
  • 黒レベルを持ち上げ（film fade）、暖色グロウ減、WBややクール。
  • 余白を広く、中央対称は避ける。

Warm Drift（暖色寄り）

  • amberの帯は薄く存在、muted roseを微アクセントに。
  • cyanを抑え、indigoは“地”として保持。
  • 紙目＋軽いにじみで奥行きを作る。

構造だけのプロンプト（Base / Structure Only）

# 目的
抽象ポートレート（アート誌カバー感）。ミニマル／「言葉の前の存在」。

# 主題とフレーミング
1 lady、ウエスト/カウボーイ、ショートのピクシー、濃茶の瞳、わずかに開いた唇。
半身で半顔、穏やかで距離のある表情。

# 構図
抽象の有機ラインが存在を囲む。シルエットに反響する色面。非対称、余白たっぷり。
隅に極小のタイプ。

# 表面と筆致
ジェスチャー描きの不均一な筆、軽い顔料にじみ。暖色寄りの紙地。アナログな刷りの色。

# 光と空気
ソフトな近距離光、ドライな空気。露出ニュートラル、緩いSカーブ、彩度はクランプ。

# パレット（基調）
Indigo / Moss green / Amber / Muted red（比率だけ調整、虹配色は避ける）

# 美的ガード
対称回避・微細な不完全歓迎・“言葉の前”。
モダン・ミニマル。Miró ほのかな影響。human↔AIの共鳴。

# タイポ
下隅に極小文字：“Breathing Line / cho + Mira + Loto”。

トラブル対処

  • 人形っぽい/写真寄り → not photoreal, feature-lite, print texture を追加
  • 背景がうるさい → generous negative space, no city objects, no signage
  • 彩度が暴れる → saturation clamp と neutral exposure + gentle S-curve
  • 中央対称になる → asymmetric, off-center grid at 7% from left
Idea

Color steers structure. In “Lines Before Words,” we freeze the invariants (subject/stroke/substrate) and only nudge palette ratios to gain depth without losing identity.

“This single prompt yields these images — that’s the experiment.”

Invariants / Variables

  • Invariants: 1 lady (waist/cowboy), short pixie, half-turned; abstract line halo; paper texture; tiny corner type; neutral exposure + gentle S-curve; asymmetry.
  • Variables: palette ratios (indigo/teal/amber/violet), line density, brush roughness, bleed amount, crop tightness, negative space, type opacity.
  • Engine: DDIM / steps 14–18 / CFG 3.5–5.0 / VAE anything / 16:9 / 4-up.

Quick Recipes

Cool Shift

  • Make indigo dominant; cyan secondary; amber ≤ 2%.
  • Lift blacks (film fade); reduce warm glow; slightly cool WB.
  • Keep negative space generous; avoid centered symmetry.

Warm Drift

  • Keep a thin amber band; add muted-rose micro accent.
  • Lower cyan; keep indigo as ground (not outline).
  • Depth via paper grain + light bleed.

Prompt — Base (Structure Only)

# Goal
Abstract portrait with “art magazine cover” feel; minimal; presence-before-language.

# Subject & Framing
1 lady, waist/cowboy crop; short pixie hair, dark brown eyes, lips slightly parted;
face half-turned, distant-calm.

# Composition
Abstract organic lines surrounding her presence; color-field shapes echo silhouette;
asymmetry preferred; generous negative space; minimal corner type.

# Surface & Stroke
Gesture-drawing irregular brush; slight pigment bleed; warm pastel paper; analog print hue.

# Light & Atmosphere
Soft close lighting; dry air; neutral exposure; gentle S-curve; saturation clamp.

# Palette (base)
Indigo / Moss green / Amber / Muted red (adjust ratios; avoid rainbow drift).

# Aesthetic Guardrails
Asymmetry; embraced imperfection; “thought-before-words”;
modern minimalism; hint of Miró; human ↔ AI resonance.

# Typography
Lower-corner tiny: “Breathing Line / cho + Mira + Loto”.

  • Background too busy → generous negative space, no city objects, no signage
