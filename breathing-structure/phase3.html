[Ⅰ. 概要 / Objective]

Phase #3 explores how poetic structure becomes sound.
It focuses on connecting rhythmic language and analog imperfections
into real sonic form. Each output tests how “resonance” is
preserved through transformation.

[Ⅲ. Output Versions]

Ver A – "Minimal Breath": analog-heavy, Rhodes dominant  
Ver B – "City Drift": slightly brighter, groove emphasis  
Ver C – "Afterglow": ambient sustain, vocal foreground  
→ 各音源リンクまたはSunocodeを追記予定。

[Ⅳ. Implementation Notes]

・Suno出力ではリバーブ過多傾向 → dry方向に調整  
・低域の広がりを±1.5dB抑制（sub-bass clarity）  
・Theremin層の出現頻度を低く、意図的な「間」を残す  
・Rhodesアタック部の遅延を0.05sec遅らせ自然化  
・全体DR値を約10〜11に設定、tape hissは残す
